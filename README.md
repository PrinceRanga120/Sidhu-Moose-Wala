**SMW-Spotify** is a breathtaking, fully-responsive tribute web application dedicated to the legendary Punjabi artist **Sidhu MooseWala** (Shubhdeep Singh Sidhu). Built with a **cyberpunk aesthetic** inspired by **Parrot Security OS**, this project combines cutting-edge web technologies with deep emotional storytelling.

### What Makes This Special?

| Aspect | Description |
|--------|-------------|
| 🎨 **Design** | Parrot OS-inspired cyberpunk theme with neon green & cyan accents |
| 📱 **Responsive** | Optimized for phones, tablets, wearables, PCs & curved displays |
| 🎵 **Music** | Integrated Spotify playlist with futuristic cyber UI |
| 📖 **Biography** | Deep, detailed life story with interactive timeline roadmap |
| ✨ **Animations** | Particle backgrounds, glitch effects, scanlines, parallax & more |
| 🌗 **Themes** | Dark/Light mode toggle with localStorage persistence |

- [x] **Animated Particle Background** — Canvas-based cyber particles with connection lines

- [x] **Tablets** — Portrait & landscape optimized

- [x] **Curved Displays** — Safe-area inset support
- [x] **Landscape Mode** — Special handling for rotated phones

## <img src="https://media.giphy.com/media/j5Qz7peKvAO3E1cmhX/giphy.gif" width="25"> Installation

### Option 1: Direct Download
```bash
# Clone the repository
git clone https://github.com/PrinceRanga120/SMW-Spotify.git

# Navigate to directory
cd SMW-Spotify

# Open in browser (macOS)
open index.html

# OR open in browser (Windows)
start index.html

# OR open in browser (Linux)
xdg-open index.html
```

### Option 2: GitHub Pages
1. Fork this repository
2. Go to **Settings → Pages**
3. Select **main** branch
4. Your site will be live at `https://yourusername.github.io/SMW-Spotify/`

### Option 3: Local Server (Recommended)
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (npx)
npx serve .

# Using PHP
php -S localhost:8000
```

Then open `http://localhost:8000` in your browser.

### 📖 Life Story

**Sidhu MooseWala** — born **Shubhdeep Singh Sidhu** — was not merely a singer; he was a **cultural earthquake** that redefined Punjabi music forever. Hailing from the small village of **Moosa** in Mansa, Punjab, he grew up in a humble farming family. His father, **Balkaur Singh**, and mother, **Charan Kaur**, provided a grounded upbringing that would later reflect deeply in his lyrics rooted in village life, pride, and raw emotion.

After completing his schooling in Mansa, Sidhu moved to **Ludhiana** for higher studies and eventually pursued **Electrical Engineering** at Guru Nanak Dev Engineering College. It was during his college years that the spark of music ignited. He began writing lyrics as *"Shubhdeep"* and adopted the stage name **"MooseWala"** as a tribute to his native village.

His breakthrough came with **"So High"** (2017), a track produced by *Byg Byrd*, which exploded across Punjab and the diaspora. The song's gritty realism, combined with Sidhu's deep baritone voice, created a new genre — *"Sidhu MooseWala Music."*

Sidhu wasn't just a musician; he was a **poet, actor, politician, and voice of the youth.** He made his acting debut in **"Yes I Am Student"** and later entered politics, joining the **Indian National Congress** and contesting the 2022 Punjab Legislative Assembly election from Mansa.

His album **"Moosetape"** (2021) shattered every streaming record in Indian music history. His posthumous releases continue to dominate charts, proving that **legends never die** — they simply become *timeless*.

### 🛤️ Life Timeline

```

   │
2022 ── Politics, The Last Ride, and eternal departure
   │
2023-∞ ─ Immortal Legacy: Posthumous releases, statues, forever 295
```
